Введение

Обучающая система по основам алгоритмизации и программирования будет использоваться для автоматизации процесса обучения школьников. Она позволяет учителю создавать и рассылать задания своим ученикам, выставлять оценки за задания и вести электронный журнал. Ученики смогут выполнять задания и отслеживать оценки.

Авторизация. Запуск веб-приложения

После перехода на сайт неавторизованный ученик увидит окно авторизации. Необходимо ввести логин и пароль, выданные ему учителем, и нажать кнопку “Войти”. Пользователь авторизуется в системе с ролью “Ученик”.

Журнал ученика

Ученик, успешно пройдя авторизацию, увидит главную страницу - журнала ученика. На этой странице ученик сможет отслеживать свои домашние задания, а также среднюю оценку и место в рейтинге класса. Для завершения работы с веб-приложением нужно нажать на кнопку “ВЫХОД”.

Рейтинг класса

Чтобы посмотреть рейтинг класса ученику необходимо нажать на кнопку “Посмотреть рейтинг класса”. Рейтинг класса мотивирует выполнять все задания качественно. Чем выше рейтинг ученика, тем выше он в рейтинге среди своих одноклассников. Значение рейтинга рассчитывается следующим образом: ЗДЕСЬНАПИСАНАФОРМУЛА. Для завершения работы со страницей рейтинга класса нужно нажать на кнопку “Вернуться к журналу”.

Решение задания

Эта страница предназначена для выполнения задания. Ученик должен составить алгоритм, чтобы исполнитель (муравей) выполнил условие задачи. Необходимо перетаскивать командные блоки из поля “команды” в поле “алгоритм”. Система автоматически составит блок-схему, алгоритму которой будет следовать исполнитель. В поле “Справка” написана основная информация о циклах и об энергии. Ученик может запускать свой алгоритм нажатием на кнопку “Play” (треугольник под полем с исполнителем). Исполнитель может собирать монеты и двигать камни, в том числе заполнять ими лунки. Однако, исполнитель не может двигать сразу два камня, выходить за поле и ходить по лункам. Чтобы отправить свой алгоритм на проверку учителю, ученик должен нажать на кнопку “Отправить на проверку!”. Для завершения работы со страницей решения задания нужно отправить задание или нажать на кнопку “Вернуться к журналу”.
